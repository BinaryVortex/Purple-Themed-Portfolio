## About

This is a clean, responsive, purple-themed personal portfolio homepage built with plain HTML, CSS, and JavaScript. It showcases a hero section, project highlights, skills, and contact links — designed to be easy to customize and deploy.

## Features

- Attractive purple color palette and modern layout
- Responsive design for desktop and mobile
- Smooth scrolling and simple JavaScript interactions
- Easy to customize — no build tools or frameworks required

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

To use this portfolio locally:

1. Clone the repository:

```bash
git clone https://github.com/BinaryVortex/Purple-Themed-Portfolio.git
```

2. Open `index.html` in your browser:

```bash
# macOS / Linux
open index.html

# Windows (PowerShell)
Start-Process index.html
```

Or serve the folder with a simple static server (recommended for testing local fetches):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```

## Customize

- Edit `index.html` to change content and sections.
- Update `styles.css` (or the stylesheet included in the project) to tweak colors, spacing, and typography.
- Add or modify JavaScript in the project file(s) to change interactions.

## Deployment

This project can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting provider. For GitHub Pages, enable Pages from the repository settings and set the source to the `main` branch.
